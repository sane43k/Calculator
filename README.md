## How to run the application

### Running the application

1. Install dependencies
   ```
   npm install
   ```
2. Build the app
   ```
   npm run build
   ```
3. Start the app
   ```
   npm start
   ```

### Running eslint

```
npm run lint
```

### Running prettier

npm run format
