## How to run the application

### Install dependencies

```
npm install
```

### Running the application

- Build the app

  ```
  npm run build
  ```

- Start the app

  ```
  npm start
  ```

### Running tests (Jest)

- Running

  ```
  npm test
  ```

- To automatically restart on changes

  ```
  npm test -- --watch
  ```

- Shows how much code is covered by tests

  ```
  npm test -- --coverage
  ```

### Running eslint

```
npm run lint
```

### Running prettier

```
npm run format
```
